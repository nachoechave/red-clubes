# Red Clubes - Arquitectura inicial

## 1. Objetivo del documento

Este documento describe la arquitectura inicial del sistema Red Clubes.

La arquitectura explica cómo se organizan las partes principales del proyecto y cómo se comunican entre sí.

El objetivo es tener una visión clara antes de comenzar a programar el backend, el frontend y la base de datos.

---

## 2. Visión general del sistema

Red Clubes será una aplicación web compuesta por tres partes principales:

- Frontend
- Backend
- Base de datos

Cada parte tiene una responsabilidad diferente.
